## Features

- Quit Touch Bar
- CPU
- Disk
- Network meter
- Spotify
- displaySleep
- volumeDown/Up
- mute
- battery
- weather
- timeButton

### Spotify

- Create `/Users/Shared/MTMR/` folder and drop there spotify.scpt from [here](https://github.com/Wonz5130/TouchBar-Presets-for-MTMR/blob/master/spotify.scpt)
