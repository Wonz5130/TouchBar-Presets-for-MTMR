## 特性

- 退出 MTMR
- CPU占用情况
- 磁盘使用情况
- 上传下载速度
- Spotify 正在播放歌曲
- 休眠
- 音量 -/+
- 静音
- 电池
- 天气
- 日期 & 时间

### Spotify

- 在 Mac 上创建 `/Users/Shared/MTMR/` 文件夹，并从[这里](https://github.com/Wonz5130/TouchBar-Presets-for-MTMR/blob/master/spotify.scpt)拷贝 spotify.scpt 文件。
